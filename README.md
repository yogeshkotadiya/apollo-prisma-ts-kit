# Apollo Prisma Typscript Starter

## Stack

- Node
- Express
- Apollo
- Prisma
- GraphQL
- Postgres

---

###Get started

### Clone the repo

```bash
git clone https://github.com/yogeshkotadiya/apollo-server-prisma-ts-starter.git
```

```bash
yarn install
```

Use Prisma deploy to connect and generate prisma TypeDefs

```bash
yarn deploy
```

Then just start the server

```bash
yarn dev
```
